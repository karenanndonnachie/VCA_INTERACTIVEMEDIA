Minimal Sketch

This is the smallest complete p5 sketch: create a canvas once in setup() and draw each frame in draw().

Use this as your default starter structure.

function setup() {
  createCanvas(400, 300);
}

function draw() {
  background(255);
  fill(0);
  circle(200, 150, 50);
}
Visual

Understanding setup() and draw()

  • setup(): Runs once at the start. Initialize canvas and variables.
  • draw(): Runs repeatedly (default 60 times/second). Update and render.

Basic Shapes

Shape functions are coordinate-based drawing tools.

Start with primitives and combine them into more complex visuals.

/* Rectangle: x, y, width, height */
rect(50, 50, 100, 80);

/* Circle: x, y, diameter */
circle(100, 100, 50);

/* Ellipse: x, y, width, height */
ellipse(200, 150, 100, 60);

/* Line: x1, y1, x2, y2 */
line(0, 0, 200, 200);

/* Triangle */
triangle(100, 10, 40, 198, 190, 170);

/* Point: x, y */
point(50, 50);
Visual

Polygon (Arbitrary Shapes)

beginShape();
vertex(30, 20);
vertex(85, 20);
vertex(85, 75);
vertex(30, 75);
endShape(CLOSE);

Shape Modes

/* CORNER: Corner is at x, y (default) */
rectMode(CORNER);

/* CENTER: Center is at x, y */
rectMode(CENTER);
rect(100, 100, 50, 50);

/* RADIUS: x, y is center, width/height are radii */
rectMode(RADIUS);

Setting Colors

fill() colors shape interiors, stroke() controls outlines, and background() clears each frame.

RGBA adds transparency with the fourth value.

/* Fill color (interior) */
fill(255, 0, 0);  /* Red */
fill(255);        /* Grayscale */
fill(255, 0, 0, 150);  /* RGBA - with alpha/opacity */

/* Stroke color (outline) */
stroke(0);

/* Stroke weight (thickness) */
strokeWeight(2);

/* Background color */
background(200);
Visual

No Fill / No Stroke

noFill();      /* Draw shapes with no fill */
noStroke();    /* Draw shapes with no outline */

/* Reset to default */
fill(0);
stroke(0);

Color Values

/* RGB: 0-255 each */
fill(255, 128, 0);

/* RGBA: with alpha 0-255 */
fill(255, 128, 0, 200);

/* HSB: Hue, Saturation, Brightness */
colorMode(HSB);
fill(hue, saturation, brightness);

Global Variables

Global variables persist between frames, which makes animation possible.

Incrementing position values in draw() creates continuous motion.

let x = 100;
let y = 150;
let speed = 5;

function setup() {
  createCanvas(400, 300);
}

function draw() {
  background(255);
  x += speed;
  circle(x, y, 30);
}
Visual

Loops

/* Draw 10 circles in a row */
for (let i = 0; i < 10; i++) {
  circle(i * 50, 100, 20);
}

/* While loop */
let count = 0;
while (count < 5) {
  console.log(count);
  count++;

Simple Animation

Animation in p5 is just repeated drawing with changing values.

Here, x increments each frame, and resets once it leaves the canvas.

let x = 0;

function draw() {
  background(255);
  fill(0);
  circle(x, 100, 30);
  
  /* Increment x each frame */
  x = x + 5;
  
  /* Reset when off screen */
  if (x > width) {
    x = 0;
  }
}
Visual

Using sin() and cos() for Motion

sin() and cos() create smooth oscillation and orbital movement.

Using frameCount as time input produces continuous motion.

function draw() {
  background(255);
  
  /* Use sine for smooth up/down */
  let y = 150 + sin(frameCount * 0.05) * 50;
  
  /* Use cosine for smooth left/right */
  let x = 200 + cos(frameCount * 0.05) * 100;
  
  circle(x, y, 30);
}
Visual

Text Output

Text uses the same drawing pipeline as shapes.

Combine textSize(), fill(), and textAlign() for readable labels and titles.

/* Set text size in pixels */
textSize(16);

/* Set text color */
fill(0);

/* Draw text at x, y */
text('Hello World', 50, 100);

/* Text alignment */
textAlign(CENTER);
text('Centered', width/2, 50);
Visual

Random Values

random() adds variation and unpredictability to sketches.

Convert to integers with int() for dice rolls, indexes, or grid selections.

/* Random decimal 0-1 */
random();

/* Random between min and max */
random(50, 100);

/* Random integer */
int(random(1, 7)); /* 1-6 like a die */
