Basic Selectors

Use selectors to target different parts of your HTML.

Element targets all tags, class targets reusable groups, ID targets one specific element, and attribute targets form fields or elements with matching attributes.

/* Element selector */
p { color: blue; }

/* Class selector */
.container { width: 100%; }

/* ID selector */
#header { background: black; }

/* Attribute selector */
input[type="text"] { border: 1px solid #000; }

Combinators

/* Descendant */
div p { color: red; }

/* Child */
ul > li { margin: 10px; }

/* Adjacent sibling */
h1 + p { font-weight: bold; }

/* General sibling */
h1 ~ p { margin-top: 20px; }

Pseudo-classes

Pseudo-classes style elements in a temporary state, like hover, focus, visited, or based on position in a list.

These are useful for interaction feedback and making UI states visible to users.

a:hover { color: yellow; }
a:visited { color: purple; }
input:focus { outline: 2px solid blue; }
li:first-child { font-weight: bold; }
p:nth-child(2n) { background: #f0f0f0; }

BOX MODEL

Understanding Box Model

Every element consists of: Content → Padding → Border → Margin

Explanation
Code Snippet
Visual Result

The box model controls spacing around every HTML element.

Padding is inside the border, margin is outside, and width applies to content unless you change sizing mode.

.box {
  /* Space inside element */
  padding: 20px;
  
  /* Outline of element */
  border: 2px solid black;
  
  /* Space outside element */
  margin: 30px;
  
  /* Width affects content area */
  width: 200px;
}

Container Properties

.container {
  display: flex;
  
  /* Direction */
  flex-direction: row; /* row | column | row-reverse | column-reverse */
  
  /* Horizontal alignment */
  justify-content: center; /* flex-start | center | space-between | space-around */
  
  /* Vertical alignment */
  align-items: center; /* flex-start | center | stretch */
  
  /* Wrap items */
  flex-wrap: wrap;
  
  /* Gap between items */
  gap: 20px;
}

Item Properties

.item {
  /* Growth factor */
  flex-grow: 1;
  
  /* Shrink factor */
  flex-shrink: 1;
  
  /* Base size */
  flex-basis: 200px;
  
  /* Shorthand */
  flex: 1 1 200px;
}

Practical Flex Pattern (Copy + Use)

This is a common responsive layout split: sidebar + main content.

On smaller screens, the media query stacks columns vertically for better readability.

.layout {
  display: flex;
  gap: 16px;
  flex-wrap: wrap;
}

.sidebar {
  flex: 1 1 220px;
}

.content {
  flex: 2 1 360px;
}

@media (max-width: 768px) {
  .layout {
    flex-direction: column;
  }
}

Basic Grid

.grid {
  display: grid;
  
  /* Define columns */
  grid-template-columns: 1fr 1fr 1fr;
  
  /* Define rows */
  grid-template-rows: 100px 200px;
  
  /* Gap between items */
  gap: 20px;
}

Responsive Grid

auto-fit with minmax() is a fast way to make card layouts responsive.

Columns automatically add/remove based on available width, while keeping each item above a minimum size.

.auto-grid {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
  gap: 20px;
}

Gradients

Gradients are image-like backgrounds generated with CSS, so no image file is needed.

Linear moves in a direction; radial spreads from a center point.

/* Linear gradient */
background: linear-gradient(90deg, #ff0000, #0000ff);

/* Radial gradient */
background: radial-gradient(circle, yellow, blue);

Transform

Transforms change an element’s position or shape without affecting surrounding layout flow.

Common transforms include translate, rotate, and scale.

.element {
  transform: translate(10px, 20px); /* Move */
  transform: rotate(45deg); /* Rotate */
  transform: scale(1.5); /* Enlarge */
  transform: skew(10deg); /* Skew */
}

Transitions

Transitions animate a change between states (like hover), while keyframe animations run over time automatically.

Use transitions for UI interactions and keyframes for repeating motion.

.button {
  background: blue;
  
  /* Animate property changes */
  transition: background 0.3s ease;
}

.button:hover {
  background: red;
}

@keyframes slide {
  from { transform: translateX(0); }
  to { transform: translateX(100px); }
}

.element {
  animation: slide 1s ease infinite;
}

Responsive Design

Media queries let styles adapt at specific viewport widths.

This example starts mobile-first, then increases container widths at tablet and desktop breakpoints.

/* Mobile first */
.container {
  width: 100%;
}

/* Tablet */
@media (min-width: 768px) {
  .container {
    width: 750px;
  }
}

/* Desktop */
@media (min-width: 1024px) {
  .container {
    width: 1000px;
  }
}
